本系统开发时的初衷：`更快 更好 更简洁 更方便 更贴近`

作者在使用一些渗透系统的时候，发现工具过于重复，累赘，很多东西甚至是不需要的工具，因此和一些小伙伴共同设计了此系统，经过两个月内测，以及优化，经过群内两百多名师傅的测试下，`JiaoSuInfoSec_T00ls_Win11_24722_v1.1.RT_hw.Beta`发布
